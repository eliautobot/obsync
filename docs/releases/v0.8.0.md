# Obsync v0.8.0

Obsync v0.8.0 turns the Windows bridge into a first-class desktop app and adds explicit control over running work and watched folders.

## Highlights

- **Obsync Desktop:** One Windows app now bundles pairing, the background folder watcher, automatic startup, **Start this PC**, **Stop this PC**, and **Open Obsync**. It replaces the separate Companion product experience.
- **Global Start/Stop:** The dashboard can stop all synchronization, cancel queued work, cancel active extraction and AI classification, and keep computers connected for status. Starting again reconciles changes that occurred while stopped.
- **Remove one folder:** Every watched folder has a safe **Remove** action. It disappears from Obsync and is forgotten by the desktop watcher, while the real folder, original files, and existing Obsidian notes are kept.
- **Safe upgrade:** Obsync Desktop preserves existing pairing, watched folders, and vault configuration while replacing the legacy `Obsync Companion` startup task.

## Windows upgrade

Update the server first. Then open **Sources → Add another computer → Download Obsync Desktop** on each Windows computer and choose **Connect and install**. A valid existing pairing is reused; no new computer is created. The app installs under the current Windows user, needs no Administrator rights, and runs its watcher without a visible terminal.

Early community builds are not code-signed. Verify the executable came from the official Obsync release before accepting a Windows SmartScreen prompt.

## Safety behavior

Stopping or removing never edits or deletes a source file. Stopping also never disconnects a computer. Removing a folder clears only Obsync's records for that root. Existing generated notes remain in the vault, including any manual text below **My notes**.
